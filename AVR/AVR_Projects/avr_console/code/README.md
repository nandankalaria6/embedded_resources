UPod AVR SmartConsole
A homemade smartconsole/UI system built on the ATmega328P with:

SSD1306 OLED display
Rotary encoder UI
Bluetooth communication (HC-05)
Time/date sync from phone
Dino runner game
Stopwatch
Dice tool
Menu-based interface

Hardware
ATmega328P @ 16MHz
SSD1306 OLED
HC-05 Bluetooth
Rotary encoder
Push buttons

Features
Home screen
Bluetooth connectivity indicator
Time + date sync
Tools menu
Dino game
Stopwatch
Expandable UI framework

Build
make

Flash
make flash

Set Fuses
make fuses
